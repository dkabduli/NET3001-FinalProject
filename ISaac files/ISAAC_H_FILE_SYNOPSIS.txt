include/Buzzer.h
What it does:
- Declares buzzer control API.
How it worls:
- Exposes Buzzer_init() and Buzzer_beep_once() declarations.
Why it matters:
- Provides simple audible alert control.

include/LCD.h
What it does:
- Declares LCD public API.
How it worls:
- Exposes LCD_init(), LCD_clear(), LCD_line1(), LCD_line2().
Why it matters:
- Lets other modules show text without LCD low-level details.

include/SevenSeg.h
What it does:
- Declares 7-segment API (shift-register based).
How it worls:
- Exposes SevenSeg_init() and SevenSeg_show_digit().
Why it matters:
- Allows numeric output with simple API calls.

include/Ultrasonic.h
What it does:
- Declares ultrasonic distance API.
How it worls:
- Exposes Ultrasound_init() and Ultrasound_read_cm().
Why it matters:
- Supplies distance input for crossing detection.

include/TrafficLight.h
What it does:
- Declares traffic controller public API.
How it worls:
- Exposes TrafficLight_init() and TrafficLight_step().
Why it matters:
- Main control interface used by loop().
