include/GPIO.h
What it does:
- Declares GPIO setup API.
How it worls:
- Exposes GPIO_init() declaration.
Why it matters:
- Keeps all pin setup centralized before modules run.

include/PCINT_buttons.h
What it does:
- Declares button interrupt/debounce API.
How it worls:
- Exposes PCINT_buttons_init(), PCINT_consume_pause_click(), PCINT_consume_reset_click().
Why it matters:
- Gives clean button click events to control logic.

include/Timer1Tick.h
What it does:
- Declares 1-second timer tick API.
How it worls:
- Exposes Timer1_init_1Hz_tick() and Timer1_consume_1s_tick().
Why it matters:
- Separates timing from traffic behavior logic.

include/TrafficLight.h
What it does:
- Declares top-level traffic controller API.
How it worls:
- Exposes TrafficLight_init() and TrafficLight_step().
Why it matters:
- Defines the main control interface used by loop().

include/USART0.h
What it does:
- Declares UART serial API.
How it worls:
- Exposes USART0_init_9600(), USART0_print(), USART0_print_u16().
Why it matters:
- Provides shared debug and event logging output.
